# Wind Forecast Notebook

This repo contains the `wind_forecast_blab.ipynb` notebook, which explores techniques for forecasting wind behaviour.

## Getting Started
- Install dependencies for your Python environment (e.g. `pip install -r requirements.txt` if available).
- Launch Jupyter Lab or Notebook: `jupyter lab` or `jupyter notebook`.
- Open `wind_forecast_blab.ipynb` and run the cells in order.
